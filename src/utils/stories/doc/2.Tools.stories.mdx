# Tools

## ESLint

Fix rules for coding, you can use .eslintrc file to configure it.
Follow the link for documentation : https://eslint.org/docs/user-guide/getting-started

## Prettier

Fix rules for code formatting. Configured to work well with eslint. Take care when changing configuration of prettier and eslint

## Babel

Transcode ES6 code to commonJS to work anywhere.
you can use .babelrc file to configure it: https://babeljs.io/docs/en/config-files

## Storybook

Allow to run components in a storybook page (atomic development).
you can use .storybook folder to configure it.<br/>
**You may read the next link, best documentation ever**<br/>
See the documentation : https://storybook.js.org/docs/react/get-started/introduction<br/>
See this documentation too : https://www.learnstorybook.com/

## Stylelint

Format CSS and SCSS code. Disable it if you don't care
you can use .stylelint file to configure it: https://stylelint.io/
