# Project Hierarchy

In react you may prefere having 100 folders in "components" folder instead of grouping it into parent folders.<br/>
Using visual studio code, you can easily access a component by search it by name using "Ctrl+P".<br/>

**Don't be afraid of big lists of folders** in Visual Studio code.
Once you are working on the good item, you may not need to search things out of it.
